Politique de confidentialité

Run by Léa — dernière mise à jour : 13 juillet 2026

Run by Léa est une application personnelle de planification d'entraînement running. Cette page explique quelles données sont collectées, pourquoi, et comment elles sont protégées.

1. Données collectées

Lors de la création d'un compte et de l'utilisation de l'application, les données suivantes peuvent être collectées :

  • Compte : adresse email et mot de passe (stocké de façon sécurisée et chiffrée par notre prestataire d'authentification, jamais en clair).
  • Profil coureur : prénom, nom, année de naissance, poids, taille, fréquence cardiaque maximale, records personnels sur différentes distances.
  • Données d'entraînement : plans générés, séances réalisées ou manquées, notes personnelles, objectifs de course.
  • Localisation : position GPS approximative, utilisée uniquement pour afficher la météo locale et les alertes de chaleur. Non stockée de façon permanente, non partagée.
  • Données Strava (optionnel) : si vous connectez votre compte Strava, l'application accède à vos activités de course (distance, allure, fréquence cardiaque) pour comparer vos performances réelles à votre plan.

2. Pourquoi ces données sont collectées

Toutes les données servent exclusivement à générer et adapter votre plan d'entraînement personnel, assurer la synchronisation entre vos appareils, et vous fournir des informations météo pertinentes pour vos sorties. Aucune donnée n'est utilisée à des fins publicitaires.

3. Stockage et sécurité

Vos données sont stockées sur Supabase (base de données PostgreSQL hébergée), avec un chiffrement en transit (HTTPS) et une isolation stricte au niveau base de données (Row Level Security) garantissant que seul vous pouvez accéder à vos propres données.

Une copie locale de certaines données est également conservée sur votre appareil (stockage local du navigateur) pour un fonctionnement hors ligne.

4. Partage avec des tiers

Vos données ne sont ni vendues, ni partagées à des fins commerciales ou publicitaires. Certaines données transitent par les services suivants, uniquement pour fournir les fonctionnalités de l'application :

  • Strava (si vous connectez votre compte) — pour récupérer vos activités de course.
  • Open-Meteo — pour la prévision météo, sur la base de votre position GPS. Ce service ne reçoit aucune donnée personnelle identifiable.
  • Anthropic (Claude) — pour générer de courts messages de coaching personnalisés. Aucune donnée de compte n'est transmise, uniquement le contexte nécessaire au message.

5. Vos droits

Vous pouvez à tout moment :

  • Consulter et modifier vos données depuis les Réglages de l'application.
  • Déconnecter votre compte Strava depuis les Réglages.
  • Demander la suppression complète de votre compte et de vos données en nous contactant (voir ci-dessous).

6. Conservation des données

Vos données sont conservées tant que votre compte est actif. En cas de suppression de compte, l'ensemble de vos données personnelles est supprimé de nos serveurs.

7. Application à usage restreint

Run by Léa est actuellement développée pour un usage personnel et familial restreint. Elle peut évoluer vers un usage plus large ; cette politique sera mise à jour en conséquence le cas échéant.
